lsecities-2012 (1.0.1) stable; urgency=low

  * [new feature] Urban Age conferences/cities menu in the Urban Age
    Conferences section of the site now support displaying a conference
    title besides city and year - e.g.
      Electric City
      London | 2012
  * sliders are now responsive
  * event pages:
    * [new feature] added support for linking to a Storify story
    * [new feature] editors can insert in advance an event blurb to be
      displayed before the event and a blurb to be displayed once the
      event has happened
    * [bug fixed] contact and booking access info was supposed to be
      displayed before an event but wasn't displaying - this is now
      fixed
